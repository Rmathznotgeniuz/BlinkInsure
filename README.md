# BlinkInsure AI 🚀
BlinkInsure AI is an AI-powered parametric insurance system designed to protect Blinkit delivery partners from income loss caused by real-world disruptions.

## Problem
Gig delivery workers face unpredictable income due to disruptions like extreme weather, platform outages, and infrastructure failures. There is currently no system to compensate their income loss during such events.

## Solution
BlinkInsure AI is an AI-powered parametric insurance platform that automatically compensates delivery partners when disruptions occur using real-time data triggers.

## Disruption Triggers
- Extreme heat
- Heavy rainfall and flooding
- Road blockages
- Platform server outages
- Dark store shutdowns
- Internet failures
- Demand collapse
- Restricted zones

## AI Features
- Predictive Disruption Shield to forecast disruptions  
- AI Worker Twin for personalized risk modeling  
- Fraud detection using multi-layer validation  

## System Workflow
1. Worker registers  
2. AI calculates risk  
3. Worker selects weekly plan  
4. System monitors disruptions  
5. Claim auto-triggered  
6. Fraud check  
7. Instant payout  

## Anti-Spoofing Strategy
The system detects fraud using behavioral patterns, device signals, network validation, and anomaly detection to prevent GPS spoofing and fake claims.

## Tech Stack
React.js, Node.js, Python, Scikit-learn, PostgreSQL, Weather API, Google Maps API, Razorpay Sandbox

## Future Scope
- Expand to all gig platforms
- Improve AI prediction accuracy
